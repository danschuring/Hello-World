CSS Table Properties

Property Description
border Sets all the border properties in one declaration. If you want the border only around the table, and not the different cells of the table, then only attribute it to the table element.
border-collapse Specifies whether or not table borders should be collapsed. Example:
table { border-collapse: collapse; }
table, td, th { border: 1px solid black; }
border-spacing Specifies the distance between the borders of adjacent cells
caption-side Specifies the placement of a table caption
width & height These properties define the width and height respectively of tables, th, td, etc.
Horizontal alignment text-align property sets the horizontal alignment. The th default is center, the td default is left.
Vertical alignment The vertical-align property sets the vertical alignment (like top, bottom, or middle) of the content in <th> or <td>.
Padding Use padding in th and td elements.
Horizontal dividers (borders) Example:
th, td { border-bottom: 1px solid #ddd; }
Hoverable Table Use the :hover selector on <tr> to highlight table rows on mouse over like this:
tr:hover {background-color: #f5f5f5;}
Striped tables For striped tables, use the nth-child() selector and add a background-color to all even (or odd) table rows like this:
tr:nth-child(even) {background-color: #f2f2f2;}
Responsive table A responsive table will display a horizontal scroll bar if the screen is too small to display the full content.
Add a container element (like <div>) with overflow-x:auto around the <table> element to make it responsive: <div style="overflow-x:auto;">
empty-cells Specifies whether or not to display borders and background on empty cells in a table
table-layout Sets the layout algorithm to be used for a table
